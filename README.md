# 📚 Light Novel Check (라노벨 체크)

A simple and intuitive Android application to track your light novel collection status. 
(내가 읽고 있는 라이트노벨의 보유 및 구매 상태를 직관적으로 관리할 수 있는 안드로이드 앱입니다.)

## ✨ Features (주요 기능)
* **Progress Tracking:** Easily mark volumes as Owned (O), Wishlist (X), or On the Way ($).
* **Dynamic Search:** Quick filtering by book titles.
* **Localization:** Supports both **English** and **Korean** based on system settings.
* **Dark Mode:** Clean and eyes-friendly dark theme implementation.
